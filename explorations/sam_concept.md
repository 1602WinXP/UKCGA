# SAM (稀疏注意力掩码) 概念探索

## 基本思路

不修改模型权重，而是通过注意力掩码影响信息流。

示例场景：
- 用户经常询问 Python 相关问题
- 希望模型在回答时自动关注 Python 最佳实践

传统方案：
- 微调模型关注特定模式

SAM 思路：
- 在注意力计算中加入用户特定的稀疏掩码
- 掩码增强与 Python 相关的注意力权重
- 掩码是稀疏的（大部分位置为 0）
